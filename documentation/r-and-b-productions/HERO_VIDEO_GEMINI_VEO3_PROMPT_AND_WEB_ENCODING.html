Purpose & encoding reference

This document gives a copy-paste prompt for Google Gemini / Veo 3 (or similar AI video tools) to generate a cinematic hero background clip for R&B Productions, intended to loop on the site hero for no more than 10 seconds per loop.

After generation, the file must be re-encoded for the web using the same principles as Home Hero Video — Encoding & Integration Guide (fc-united/HOME_HERO_VIDEO_ENCODING_GUIDE.html): H.264, 1920×1080, yuv420p, silent AAC stereo (not video-only) for reliable Chrome playback, and +faststart.

Web delivery targets (align with encoding guide)

Setting Value / note
Duration (generation) ≤ 10 seconds per loop (use tool’s closest supported length if Veo caps at 8s, etc.).
Final encode resolution 1920×1080 — scale to cover, then center crop if source aspect differs (see ffmpeg below).
Video codec / pixel format libx264, yuv420p, -crf 23, -preset medium
Audio Silent AAC stereo (mux with anullsrc) — page plays muted; avoids Chrome freeze on video-only MP4.
Container MP4 with -movflags +faststart
Playback Hero player: autoPlay, muted, playsInline; loop via ended → seek 0 + play() (see guide).

Primary hero image prompt (16:9)

Set tool aspect to 16:9 (e.g. 1920×1080). Click 📋 Copy, then paste into Nano Banana Pro.

Ultra-wide cinematic hero image for a Malayalam–global movie production company, “R&B Productions”. Mood: soulful, premium, contemporary — not a cheesy stock poster.

Story roles (stylized, cinematic — not a crowded cast poster): In the same frame, suggest two adult partners who own the company (man in a sharp suit; woman in elegant Indian churidar / contemporary ethnic wear) — warm, confident, approachable — positioned as producers / studio heads. Separately in the scene, suggest a film director at work: on-set energy — director near a camera / monitor glow or viewfinder light, creative authority, Malayalam cinema prestige.

Featured film line (subtle, elegant typography or signage — not a full poster): “Howie MP” as the current movie the company is pushing — integrated as minimal title treatment or soft neon / etched glass on a studio element.

Brand: incorporate the R&B Productions identity as a clean logo lockup (wordmark with restrained accent), high readability, premium finish.

Lighting / grade: gold–amber key with deep teal shadows, soft volumetric haze, gentle lens bloom, 35mm film grain (subtle).

Setting: modern film studio / stage with controlled practical lights, black void + highlights (avoid clutter). Optional background hint: very subtle epic scale (distant silhouettes / mist) only if it stays tasteful — no disaster imagery, no graphic violence.

Composition: rule of thirds, clear headline space at top or left for web text overlay, not busy.

Quality: 8K feel, sharp, color-managed, high dynamic range, no watermark.

If Veo says “blocked due to safety reasons”

Image checks and text checks are separate. A message like “there are no children in the images” only means the image pipeline did not flag minors — it does not guarantee the written prompt will pass. The text layer may still block for other rules.

Common text triggers:

  • Named real people (actors, directors, owners by name) — often blocked as public-figure / likeness risk.
  • “Family-run” or similar — occasionally misfires on safety classifiers.
  • Reference photos of real individuals — can trigger likeness policies even when no minors appear.

What to try (in order):

  1. Use the safety-friendly prompt below (no celebrity names; generic adult professionals).
  2. Generate text-to-video only first (no reference images). If that works, optionally refine.
  3. If using one image, try logo-only (roy_and_bindu_prod_logo_transparent.png) — avoid face reference images until the text prompt is accepted.
  4. Add credits / real names in post (Resolve / Premiere / on-site HTML), not inside the AI prompt.
Safety-friendly prompt — use when the primary prompt is blocked

No real-person names; generic adults only; fictional brand wording. Duration ≤ 10 s. Use 📋 Copy or select all below.

Create a premium cinematic hero video (MAXIMUM 10 SECONDS) for a fictional independent film and digital entertainment label called “R&B Productions,” inspired by South Indian / Malayalam cinema culture — stylized, not documentary.

OPENING (FIRST ~1–2 SECONDS): Classic cinema “picture start” feel — a very short film-style countdown (bold numbers counting down) with a filmstrip / 35mm film reel rolling or scrolling beside or behind the numbers, subtle projector flicker, warm grading; then transition smoothly into the main sequence.

VISUAL THEME: Theatrical spotlights, warm gold and deep teal color grade, soft anamorphic lens flares, very light 35mm film grain, slow elegant camera moves (dolly / parallax). High-end, soulful, globally polished. No horror, no violence, no gore, no disturbing imagery.

PEOPLE (ADULT PROFESSIONALS ONLY — GENERIC, NOT REAL INDIVIDUALS): Three adult principals — two co-owners/producers (man in sharp business suit, woman in elegant professional attire) and one director figure — shown as the producing and directing team behind the film, representing the label. Mid-sequence: modern film office beats (monitor review, storyboard, camera silhouette). CLOSING BEAT: all three come together and stand side by side, confident and united, as the face of the brand — “making it happen,” tasteful and cinematic, not a documentary.

SETTING: Contemporary post-production suite, soft bokeh, storyboards, editing bay glow. Optional subtle modern city skyline bokeh — keep geography nonspecific unless your tool allows it.

OPTIONAL ON-SCREEN TEXT (VERY SUBTLE): Generic inspirational line only, e.g. “Soulful stories from Kerala to the world” — low contrast, brief, not dominant.

LOOP: If a seamless hero loop is required, favor matching brightness between the post-countdown frame and the final three-person pose, or use a soft crossfade in post; otherwise prioritize the countdown opener and closing brand beat.

STRICT AVOID: Celebrity names, requests to depict real public figures, graphic content, political or religious symbols, crowded chaotic scenes, neon gamer aesthetics, shaky handheld chaos.

OUTPUT: 16:9 cinematic composition, filmic contrast, clean highlights.
After a successful render: Add “An initiative by Roy and Bindu,” partner credits, and Brisbane / Malayalam production context on the website or in title cards edited in post — not inside Veo if that retriggers safety.
Primary Gemini Veo 3 prompt (copy all)

Use when not blocked. Paste into Veo 3 (or similar). Set duration to maximum 10 seconds or the closest allowed. Enable loop if the product offers it.

Create a premium cinematic hero video (MAXIMUM 10 SECONDS) for “R&B Productions,” a Malayalam-focused digital entertainment and film company.

OPENING (FIRST ~1–2 SECONDS): Film-style “showtime” opener — a short, punchy classic cinema countdown (large bold numbers: 3, 2, 1 or similar) with a filmstrip and/or 35mm film reel rolling or feeding through the frame (vintage projector aesthetic, subtle motion blur, warm gold/teal light). Keep it brief; then dissolve or cut into the main hero sequence.

THEME & MOOD: Big-screen movie atmosphere — subtle theatre spotlights, warm gold and deep teal color grade, soft anamorphic lens flares, very light 35mm film grain, elegant slow camera moves (dolly / parallax). Modern, soulful, globally polished — not a soap opera look. No horror, no violence, no clutter.

PRINCIPALS & BRAND (story, not documentary): Roy, Bindu, and Nadirshah are the film’s producers and directors — the creative leadership behind the movie and the face of R&B Productions. Roy and Bindu are partners and owners/producers. Nadirshah is the Malayalam film director on the project. Depict them as confident, approachable professionals in a contemporary film-office / studio environment (editing suite silhouettes, storyboard wall, soft bokeh lights). Roy must wear a sharp suit or refined professional business attire. Bindu in elegant professional attire. Mid-sequence: collaborative beats — reviewing a frame, nodding approval, director at viewfinder or monitor (stylized silhouette or medium shots; avoid uncanny celebrity close-ups unless your tool is cleared for reference likeness).

CLOSING BEAT (FINAL ~2–3 SECONDS): All three — Roy, Bindu, and Nadirshah — come together in the same shot, standing side by side (or a slight staggered row), facing camera or three-quarter, warm and assured — representing the production company brand and “making it happen.” Hero pose, cinematic, not stiff; this is the brand lockup moment.

OPTIONAL BACKDROP HINT: Subtle suggestion of Brisbane / Australia as home base (soft skyline bokeh or daylight warmth) — keep it minimal so Kerala / cinema identity stays primary.

CREATIVE NETWORK (context only, not all on screen at once): The company collaborates with acclaimed Malayalam talent — directors Dileesh Pothan and Nadirshah, actor Fahadh Faasil, scenarist Syam Pushkaran — express this as “prestige Malayalam cinema ecosystem” lighting and production value, not a crowded group poster.

BRAND MESSAGES (optional, subtle): If text appears, use sparingly and elegant typography: “An initiative by Roy and Bindu” and/or “Digital entertainment with soulful stories — from Kerala to the world.” Low contrast, brief, do not dominate the frame.

LOOP: Prefer a gentle crossfade or match the end group pose brightness to the post-countdown opening if you need a seamless website loop; if the countdown and final pose differ too much, prioritize the narrative arc and trim or crossfade in post.

STYLE NEGATIVES: No shaky handheld chaos, no neon gamer UI, no cheap stock transitions, no cluttered subtitles, no religious or political symbols, no gore.

OUTPUT: 16:9 cinematic composition, high dynamic range feel, clean highlights, filmic contrast.

Negative / safety add-ons (append if supported)

Avoid: graphic content, hateful imagery, distorted faces, watermarks, random logos, unreadable tiny text, flicker epilepsy risk, loud strobe.
Tool limits: If Veo 3 caps clips at 8 seconds, generate at the max length, then loop in the player. If “loop” mode produces a longer file, trim to 10 seconds in ffmpeg before deploy.
Post-process: ffmpeg (10s, 1080p, silent AAC, faststart)

Replace INPUT.mp4 with your exported Veo file path. Output path targets public/images/hero_movie_section/ — adjust filenames to match your repo. 

ffmpeg -y -i "INPUT.mp4" -t 10 -vf "scale=1920:1080:force_original_aspect_ratio=increase,crop=1920:1080" -c:v libx264 -pix_fmt yuv420p -crf 23 -preset medium -f lavfi -i anullsrc=channel_layout=stereo:sample_rate=44100 -map 0:v:0 -map 1:a:0 -c:a aac -b:a 96k -shortest -movflags +faststart "public/images/hero_movie_section/rb-hero-loop.mp4"

Example from repository root on Windows (PowerShell), if the export is saved as public/images/hero_movie_section/veo3-raw-export.mp4: 

ffmpeg -y -i "public/images/hero_movie_section/veo3-raw-export.mp4" -t 10 -vf "scale=1920:1080:force_original_aspect_ratio=increase,crop=1920:1080" -c:v libx264 -pix_fmt yuv420p -crf 23 -preset medium -f lavfi -i anullsrc=channel_layout=stereo:sample_rate=44100 -map 0:v:0 -map 1:a:0 -c:a aac -b:a 96k -shortest -movflags +faststart "public/images/hero_movie_section/rb-hero-loop.mp4"
